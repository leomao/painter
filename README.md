# NMLab Web Painter
網頁版的簡易小畫家

Live Demo: http://leomao.github.io/painter

Note: 可能不相容於 IE8

## Features
各個工具都可以改顏色，線條粗細  
可以復原跟清空畫布  
另外可以下載畫好的圖  

### Tools
+ 矩形選取
+ 橢圓形選取
+ 鉛筆
+ 直線
+ 矩形
+ 橢圓形
+ 油漆桶

Bug-free(not sure?)

## To Contribute
### 事前準備
需要ruby 的 gem 跟 [bundler](http://bundler.io/)
```
$ gem install --path .bundle
```
